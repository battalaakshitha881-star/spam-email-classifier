# Spam Email Classifier 🚨

A Machine Learning project that classifies SMS messages as **Spam** or **Ham (Not Spam)**.

## 🔧 Tech Stack
- Python
- Pandas
- Scikit-learn (Naive Bayes)
- TF-IDF Vectorizer

## 📊 Results
- **Accuracy: 96%**
- Precision: 1.00 for spam detection

## 📁 Dataset
SMS Spam Collection Dataset from Kaggle (5,574 messages)

## 🚀 How to Run
```bash
python spam_classifier.py
